Assignment: Resume Using Skeleton

Description: Making a responsive Resume using the Skeleton Boilerplate css

Reflection:

Skeleton is great for making responsive, mobile-friendly websites. Skeleton can help you quickly develop websites that look great at any size. A great part about Skeleton is that is extremely easy to learn and use. For me it was easy to center and format content using the grid system. It's very easy to create a custom formatted website. All you have to do is use the container and row class, then use the preset classes to specify where certain content will be. Then it automatically resizes the content for you based on screen size. It also automatically styles elements such as button, table, and form to look much better than if you were just using html. Using skeleton is easier than using media quieries because the skeleton css grid system class automatically resizies content. It is really convienient. You dont have to worry about creating classes and divs and then edit them each individually when styling in different media quieres. However some of its weaknesses is that it does seem better for smaller projects. Adittonally, updating may cause conflicts with your code. And if you dont update, you may risk falling behind. Also, Skeleton has a limited number of templates, components and styles. And skeleton doesn't give any javascript.
